# KCast
四角いバーチャル配信アプリ



### 使い方
UnityからKCastフォルダを開けばたぶん起動します。  
UnityエディタでさらにKCast.unity(Unityアイコンのやつ)をクリックすると開けます。



中は好き勝手に変更して使ってOKです。



### キーボードでの操作
スペースキー：回転の開始/停止　

↑↓←→キー：上下左右の移動　　

##### altキーを押しながら
↑↓キー：手前 奥への移動　

←→キー：振動の大きさと速さを同時に変える　

スペースキー：初期位置に戻す　　

##### ctrlもしくはcmdキーを押しながら
↑↓キー：振動の大きさを同時に変える　

←→キー：振動の大きさと速さを同時に変える　　


##### shiftキーを押しながら
スペースキー：振動を止める　

↑↓キー：振動の速さを同時に変える　

←→キー：回転の向きを変える(押せば押すほど速く回る)
